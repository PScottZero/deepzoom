![Screenshot 1](screenshots/zoom1.jpg)
![Screenshot 2](screenshots/zoom2.jpg)
![Screenshot 3](screenshots/zoom3.jpg)

Setup:

```
pip install -r requirements.txt
```

Create deepzoom image:

```
python create.py path/to/image path/to/destination
```

View deepzoom image:

```
python deepzoom.py path/to/deepzoom
```
